学习目标

  1. 说出参考系是什么、为什么"运动"必须先选参考系，并能在同一情境下选 2 种不同参考系描述同一物体的运动
  2. 判断哪些情境可以把物体抽象为质点、哪些不可以；并能为"不可以"的情境给出物理理由
  3. 区分位移 \(\vec{x}\) 与路程 \(s\)、平均速度 \(\bar{v}\) 与平均速率，并解释为什么往返运动中 \(\bar{v}\) 可能为 0 而平均速率不为 0
  4. 说出瞬时量与平均量的差异，并能从"时间间隔越来越小"的语言出发，描述瞬时速度的直观含义（不要求微积分形式定义）
  5. 写出位移、速度、加速度的定义式，标清楚每一项的单位 / 矢量性 / 方向规定
  6. 读懂一张简单的 v-t 图与 a-t 图：能从图像直接说出"现在的运动是什么样的"（加速 / 减速 / 匀速 / 反向），不必做积分
驱动问题

站在站台上，一列高铁正在驶离。你的同伴问你："车厢里那个坐着喝水的乘客，到底是静止的还是运动的？" 你想说"他静止"——因为他没动；你又想说"他在运动"——因为高铁在跑。哪个说法对？

再往下问：你用秒表测了一段路，跑步往返了 100 米总共花 25 秒。你的速度是多少？是 \(100/25 = 4\) m/s？还是 0？还是别的什么？

"运动到底是什么"这个最朴素的问题，需要 5 个概念才能讲清楚——参考系 / 质点 / 位移 / 速度 / 加速度。本课要做的事情是把这 5 个概念逐一从"日常直觉"翻译成"可以写在纸上、可以代入公式、可以画在图上"的物理语言。

（提示：本课不展开微积分推导。瞬时速度的严格定义会在下一课"极限的物理引入"展开；本课只把"瞬时 vs 平均"的差异在直觉与图像上立起来。）

Card 1 · 参考系与质点 · 把"动"放进一个可计算的舞台

▼
学习目标：理解"运动是相对的"——任何运动描述都必须先选一个参考系；掌握把真实物体抽象为质点的两个判断准则，并能为不能抽象的情形说出原因。
建议时长：25-30 分钟
STEP 1先自己想
5-8 分钟 · 不查资料

合上书与手机。在草稿纸上独立尝试以下两件事：

  1. 用一句话描述"高铁里坐着喝水的乘客是静止还是运动"——你的判断 + 你的理由。
  2. 地球绕太阳一年转一圈，半径约 1.5 亿公里。研究"地球绕太阳的周期"时，能不能把地球看作一个"点"？为什么？研究"地球上不同纬度白昼时长差异"时呢？
写下来的答案不必"对"，只要是你自己的判断。后面 STEP 2 会把"日常直觉"翻译成物理语言——届时你回头看自己 STEP 1 的回答，能更准确地感到"我之前差在哪里"。
STEP 2参考系 · 把"动"放进一个坐标系
和 AI 一起把直觉翻译成定义
骨架 · 参考系（Reference Frame）的物理定义
定义：参考系是"假定不动、用来描述其他物体位置和运动"的物体（或物体系统）+ 固定在其上的坐标轴 + 计时起点。
"假定不动"是关键：参考系本身的运动状态不进入描述。你站在地面说"高铁向东 300 km/h 跑"，是把地面当作"假定不动"。坐高铁里的乘客说"窗外的电线杆向西 300 km/h 跑"，是把车厢当作"假定不动"。两个说法都对——它们各自相对于自己的参考系都正确。
结论 1（运动的相对性）：物体是否运动、运动多快、朝什么方向，都依赖于所选参考系。"动"或"不动"不是物体本身的属性，而是"物体 + 参考系"这一对关系的属性。
结论 2（参考系的选择是自由但有讲究的）：物理上没有"绝对正确"的参考系；但有"方便"与"不方便"的差别。研究地面附近物体的运动，通常选地面或建筑物当参考系；研究行星运动，通常选太阳当参考系——选得"巧"，定义式才简洁。
思考点 1 · 同一个动作，两个参考系下的两种描述
你站在静止的电梯里，电梯突然向上加速启动。以电梯地板为参考系，你的脚有没有"运动"？以电梯井外的楼层为参考系呢？
"我想让你帮我把'我的脚相对电梯地板没动 / 相对楼层在向上运动'这两句话用'参考系'的语言重新表述，不要给我答案，只问我一句：在两个表述里，分别把哪个物体当成'假定不动'？请你引导我自己回答。"
在草稿上写：参考系 A = ____；参考系 B = ____；同一只脚在 A 中的运动状态 = ____，在 B 中的运动状态 = ____。
卡住的话：把"假定不动"四个字反复念一遍，看哪个物体在每个表述里被默认为"舞台"。
思考点 2 · 参考系能不能"随便选"？
如果让你研究"乒乓球在桌上的弹跳"，你能不能选"乒乓球本身"作为参考系？这样会发生什么？
"不要告诉我答案。请引导我思考：如果把乒乓球本身当作'假定不动'，那么乒乓球相对自己的位置会是什么？这个描述能不能用来研究'乒乓球在桌上的弹跳'？"
在草稿上写一句话总结：参考系可以"随便选"吗？理论上 vs 实用上分别如何？
STEP 3质点 · 何时把物体看成一个点
从"为什么要抽象"开始
这一节的驱动问题

我们说"汽车向东行驶 60 km"——但汽车有 4.5 米长。是车头前进了 60 km？还是车尾？还是重心？

如果不做任何简化，"位置""速度""加速度"这些概念，能不能在一个有形状、有内部结构的物体上一句话讲清楚？

思考点 10 · 不用"质点"这个抽象，会遇到什么困难
我说"汽车在 \(t=0\) 时位置 \(x=0\)、在 \(t=10\) s 时位置 \(x=200\) m"——如果汽车没有被抽象成一个点，这两个"位置"到底指汽车上的哪一部分？
"不要直接告诉我答案。请引导我探究：如果车头、车尾、重心给出的位置数值都不一样，我们说的'汽车位置'到底是谁的位置？在这种含糊之下，我还能不能严格地算出'速度''加速度'？"
卡住的话：想象把汽车换成一整列 30 节高铁列车——你说"高铁到北京南站了"，指的是哪一节车厢？为什么这在日常语言里没歧义，但在物理公式里就出了问题？
在草稿上写一句话总结："引入质点这个抽象的动机 = ________"。
质点（Particle / Mass Point）的物理定义
定义：质点是"具有质量、没有大小和形状"的理想物理模型。把真实物体抽象成质点 = 忽略它的形状、大小、内部结构、转动，只关心它的"位置随时间如何变化"。
意义：抽象为质点后，"位置"就是一个明确的坐标点，"速度""加速度"才能被严格定义、计算、画图。质点让运动学从"含糊的日常语言"变成"可以写在纸上、代进公式的物理语言"。
判断准则 1（尺度准则）：物体自身的尺寸 \(\ll\) 它运动经过的距离时，可视为质点。地球公转研究中，地球半径约 6400 km，公转轨道半径约 \(1.5 \times 10^8\) km，前者只有后者的约 \(4\times 10^{-5}\)——可视为质点。
判断准则 2（关注准则）：研究的问题不涉及物体的形状、大小、内部转动时，可视为质点。同一辆汽车，研究"它在高速公路上的位移"——可视为质点；研究"它车轮的滚动 / 重心的高度变化 / 翻车时的姿态"——不可视为质点。
关键认识：质点不是"小物体"的代名词；地球虽然巨大，研究公转时也可视为质点；一支铅笔虽然小，研究它"被推动时的旋转"时却不能视为质点。质点是问题决定的，不是物体决定的。
思考点 11 · 同一物体，为什么有时是质点、有时不是——判据的深化
一位花样滑冰运动员在冰面上滑行。研究她"从场地一端滑到另一端的路径"——能不能视为质点？研究她"在原地做的三周跳"——能不能视为质点？同一个人，两种情形，判断为什么会变？
"不要直接告诉我答案。请引导我探究：'路径'这个描述涉不涉及她身体的形状？'三周跳'这个描述涉不涉及她身体的转动？两个描述关注的物理量本质上有什么不同，让'能否视为质点'的判断改变了？"
对回"判断准则 2 · 关注准则"—— 决定的不是"物体本身"，而是"这个研究问题关注什么"。
写下你的判断 + 理由：滑行路径 → 质点 / 非质点，因为 ________；三周跳 → 质点 / 非质点，因为 ________。
质点抽象的过程：把"有形状、有车轮、有内部结构"的真实汽车抽象成"只剩一个有质量的点"，放进参考系坐标内只关心它的位置。

Card 2 · 位移与路程 · 矢量与标量的第一次正面碰撞

▼
学习目标：理解位移是矢量、路程是标量；能用"起点 → 终点直线箭头"画出位移，能用"实际走过轨迹的总长"算出路程；理解为什么往返运动中位移可以为 0 而路程不为 0。
建议时长：30-35 分钟
STEP 1先自己想
8-10 分钟 · 拿尺子量

请你做一件真的事情（不是想象）：

  1. 在你的房间或教室找一个起点（贴一张纸条标"起点"）。从起点走到 5 米外的某个位置（"中转"），停一下；再走回起点。
  2. 用米尺或软尺量两件事——(a) 整个过程你实际走过的总长度（米）；(b) 起点到你"最终位置"的直线距离（米）+ 方向（"原地" 也是答案）。
  3. 把 (a) 和 (b) 写在草稿上。它们一样吗？为什么？
如果你的"最终位置 = 起点"，那么 (a) 是 10 米左右、(b) 是 0 米——这两个数差得很大。这种"差距"，就是路程和位移的本质差别。
STEP 2位移 · 起点指向终点的有向线段
矢量定义 + 定义式
位移（Displacement · 矢量）的定义
定义：位移是"从初始位置指向最终位置"的有向线段，记作 \(\vec{x}\) 或 \(\Delta \vec{x}\)。在直线运动中常简记为 \(x\)，但必须事先规定正方向。
定义式：\(\Delta \vec{x} = \vec{x}_{\text{终}} - \vec{x}_{\text{始}}\)。在直线运动 + 已规定正方向（如"向东为正"）的语境下：\(\Delta x = x_{\text{终}} - x_{\text{始}}\)，其中 \(x_{\text{终}}\)、\(x_{\text{始}}\) 是带正负号的坐标。
单位：米（m）。
矢量性：位移有大小（线段长）+ 方向（箭头指向）。规定正方向后，方向通过"正负号"表达——\(\Delta x = +3\) m 表示位移大小 3 m、方向沿正方向；\(\Delta x = -3\) m 表示位移大小 3 m、方向沿负方向。
关键认识：位移只看起点和终点，不看路径。哪怕你绕地球转一圈再回到起点，位移仍然是 0。
思考点 3 · 在直线运动中，位移的"方向"靠什么表达？
直线运动只有两个可能方向（"正"或"负"）。这两个方向怎么写进位移这个量？为什么不需要画三维箭头？
"不要告诉我答案。请引导我对照'规定正方向 → 用正负号代替箭头'这条线索：在二维 / 三维中位移要用箭头表达方向；在一维直线运动中，箭头能不能用正负号来替代？"
在草稿上写一句话总结：直线运动中位移的方向通过 ____ 表达；当 \(\Delta x > 0\) 时方向 ____；当 \(\Delta x < 0\) 时方向 ____。
STEP 3路程 · 实际轨迹的总长
标量定义 + 与位移的关系
路程（Distance Traveled · 标量）的定义
定义：路程是物体实际走过轨迹的总长度，记作 \(s\)。
定义式：\(s = \) 物体实际走过路径每一小段长度的总和。在直线运动中，无论方向如何，路程只累加正值——"向东走 3 m" 累加 3 m；紧接着 "向西走 5 m" 仍累加 5 m，总路程 = 8 m。
单位：米（m）。
标量性：路程只有大小、没有方向，永远 \(s \geq 0\)。
与位移的关系：① 单向直线运动（一路向东不掉头）：路程 = 位移大小；② 有掉头的直线运动：路程 \(\geq\) 位移大小（严格大于，除非物体没动）；③ 回到起点：位移 = 0，但路程 = 全程总长。
直线往返运动示意：先向东 9 m（上方蓝弧）再向西 6 m（下方金弧）。两条弧线只用于在视觉上分开两段运动，物理上仍是一维直线运动。总路程 s = 9 + 6 = 15 m；总位移 Δx = +3 m（仅看起点与终点）。
思考点 4 · 路程能不能是负数？
位移有正负，路程有没有正负？如果你向西走 5 m，"路程"是 +5 m、−5 m，还是别的？
"不要直接给我答案。请引导我对照'方向是哪个量的属性'这条线索：位移有方向（有正负），路程是标量（只有大小）；'向西走 5 m'的路程应该写成什么？"
在草稿上写：向西走 5 m，位移 = ____ m（含正负），路程 = ____ m（不含正负）。
卡住的话：把"路程是标量"与"标量没有方向"两句话默念一遍。
STEP 4矢量与标量 · 分类的深意
方向的意义 + 时间之箭的悖论
思考点 12 · 为什么要发明"矢量"和"标量"这两个大类——方向对物理量的意义
如果所有物理量都当作"数"来处理，会在哪一步出错？请对比两个式子——"位移 3 m 向东 + 位移 3 m 向西 = ？" 和 "路程 3 m + 路程 3 m = ？"。两个都是"3 + 3"，什么情况下答案不是 6？
"不要直接告诉我答案。请引导我探究：把这两次操作画在数轴上——位移的两次箭头是不是首尾相接？相接之后箭头总长是多少？路程的两次累加，为什么不需要看方向？'矢量'和'标量'这两个词，本质上是在给哪一件事分类？"
把"矢量/标量"翻译成一句话：方向决定了运算规则。矢量遵循矢量加法（大小 + 方向共同决定合成，可能相消）；标量直接相加。分类不是命名习惯，是运算规则的分野。
在草稿上写：为什么物理要发明"矢量"和"标量"两个大类 = ________（一句话回答）。
思考点 13 · "有方向感就是矢量"吗——时间之箭的悖论
时间从"过去 → 未来"有明确的"方向感"（时间之箭 · 不可逆）。那时间是矢量吗？温度从"高 → 低"也有方向感，温度是矢量吗？如果按"有方向就是矢量"的直觉判，这两个都该是矢量——但物理里它们都是标量。为什么？
"不要直接告诉我答案。请引导我探究：'早上 8 点 + 3 小时 = 11 点''早上 8 点 − 2 小时 = 6 点'——这里的'3 小时''2 小时'能不能像位移那样，画在空间坐标轴上、并做矢量加法（平行四边形或首尾相接）？如果不能，'时间的方向'和'位移的方向'是不是同一种'方向'？"
对照温度：'从 30 ℃ 变到 20 ℃' 的'方向感'来自数值大小的差，不是空间中的方位。判断矢量的关键，不是"直觉上有方向感"——是**什么？**
在草稿上写：时间是标量，因为 ________；温度是标量，因为 ________。
深化认识 · 矢量的两条充要判据
判据 1：物理量具有可在空间中指出的方向（东/西、上/下、正/负指向），不是"数值大小的方向感"，也不是"时间的单向流"。
判据 2：物理量遵循矢量加法——大小 + 方向共同决定合成结果，两个矢量首尾相接（或平行四边形）合成，可能相消（如"向东 3 m + 向西 3 m = 0"）。
关键认识：具有"方向感" ≠ 是矢量。时间、温度、质量、能量、路程都有各自的"变化方向感"，但都不是空间方向、也不遵循矢量加法——所以都是标量。物理里的"方向"专指空间方向（可选的坐标轴指向），不是"直觉上的先后 / 大小 / 因果"。

Card 3 · 速度与速率 · 瞬时与平均

▼
学习目标：写出平均速度、瞬时速度的定义式；区分速度（矢量）与速率（标量）；理解"时间间隔越来越小"如何把"平均速度"变成"瞬时速度"——但不展开微积分定义（留给下一课）。
建议时长：30-35 分钟
STEP 1先自己想
5-8 分钟

你回去看 Card 2 STEP 1 那段你"实际走过的运动"——往返一次，比如 10 m 路程、0 m 位移、25 秒。在草稿上回答：

  1. "你的速度"是多少？写出你脑子里第一个反应。
  2. 这个数字是路程除以时间，还是位移除以时间？两种算法分别得到什么数？
  3. 这两个数哪个更能告诉别人"你跑得多快"？哪个更能告诉别人"你最终走到哪里"？
STEP 2平均速度 vs 平均速率 · 矢量与标量的第二次正面碰撞
定义式 + 单位
平均速度（Average Velocity · 矢量）
文字定义：速度是描述物体位置变化快慢（和方向）的物理量。"平均速度"是这一定义在一段时间上的版本——用位移除以所用时间。
翻译成数学语言：把"位移"写成 \(\Delta \vec{x}\)、"时间"写成 \(\Delta t\)——上面这句中文就变成下面的定义式。
定义式：\(\bar{\vec{v}} = \dfrac{\Delta \vec{x}}{\Delta t}\)；在直线运动 + 规定正方向后：\(\bar{v} = \dfrac{\Delta x}{\Delta t}\)。
单位：m/s（米每秒）。
矢量性：方向与位移 \(\Delta \vec{x}\) 同向（因为 \(\Delta t > 0\)）。在直线运动中通过 \(\bar{v}\) 的正负号表达方向。
关键认识：平均速度只看起点和终点，不看中途的快慢起伏。一个人慢慢散步 5 m 再快跑 5 m 再走回起点，与原地不动 25 秒，两者的平均速度都是 0。
平均速率（Average Speed · 标量）
文字定义：速率是描述物体路程变化快慢的标量。"平均速率"是这一定义在一段时间上的版本——用路程除以所用时间。
翻译成数学语言：把"路程"写成 \(s\)、"时间"写成 \(\Delta t\)——上面这句中文就变成下面的定义式。
定义式：平均速率 \(= \dfrac{s}{\Delta t}\)，其中 \(s\) 是路程。
单位：m/s。
标量性：只有大小，没有方向，永远 \(\geq 0\)。
与平均速度的关系：① 单向直线运动：平均速度大小 = 平均速率；② 有掉头：平均速率 \(\geq\) 平均速度大小（往返运动中常严格大于）；③ 回到起点：平均速度 = 0，平均速率 = 总路程 / 总时间。
思考点 5 · 25 秒里你走了 10 m 总路程、终点回到起点。两种"速度"分别是多少？
用你 STEP 1 的实际数据：路程 \(s = 10\) m，位移 \(\Delta x = 0\) m，时间 \(\Delta t = 25\) s。算两个数——平均速度 \(\bar{v}\) 和平均速率分别是多少？
"不要给我答案。请引导我：把上面两个定义式（速度 = 位移 / 时间 · 速率 = 路程 / 时间）分别代入，得到的两个数字差距大不大？这两个数字各自更能描述运动的哪一面？"
在草稿上写两行：\(\bar{v} = \) ____ m/s；平均速率 = ____ m/s。
STEP 3瞬时速度 · 当 Δt 越来越小
直觉先于定义
瞬时速度（Instantaneous Velocity · 矢量）的直觉
问题：平均速度告诉你"一段时间里的平均快慢"；但生活里我们常常问"这一刻你跑多快"——比如雷达测速、车速表读数、撞击的瞬间速度。"这一刻"对应一个时间点，不是一段时间——平均速度的定义式 \(\Delta x / \Delta t\) 在 \(\Delta t = 0\) 时变成 \(0/0\)，没法直接算。
直觉思路：把 \(\Delta t\) 取得越来越小——从 1 秒缩到 0.1 秒、0.01 秒、0.001 秒——同时缩短的还有 \(\Delta x\)。\(\Delta x / \Delta t\) 这个比值趋近一个确定的数。这个"趋近的数"就是瞬时速度。
记号：\(v = \lim\limits_{\Delta t \to 0} \dfrac{\Delta x}{\Delta t}\)（"极限"的物理引入会在下一课展开）。
瞬时速率：瞬时速度的大小（绝对值）。两者的关系比"平均速度 vs 平均速率"简单很多——瞬时速率 = \(|v|\)。
⚠️ 跨主题前置 · 极限（蓝诺正式板块在 02_极限_物理引入）
"瞬时速度 = 平均速度在 \(\Delta t \to 0\) 时的极限"——这里出现的"极限"是数学上的核心概念，本课不展开，只在直觉与图像层面引入。下一课《02_极限_物理引入》会从"\(\Delta t\) 一连串变小，比值越来越稳"的具体计算入手，把极限的严格含义讲清楚。本课只需要你接受："\(\Delta t \to 0\) 时 \(\Delta x / \Delta t\) 趋近一个确定的数"。
思考点 6 · 为什么"瞬时"不能直接用 Δx / Δt 代 Δt = 0？
如果直接令 \(\Delta t = 0\)，分母为 0，没意义。可是"\(\Delta t\) 趋近 0"为什么就有意义？
"不要直接告诉我极限的定义。请引导我用一个具体场景思考：假设物体位移 \(x(t) = 5 t^2\)，分别取 \(\Delta t = 1, 0.1, 0.01\) 三种情况，算出 \(t = 2\) 附近的平均速度。三个数是不是越来越接近一个固定值？这个固定值意味着什么？"
在草稿上把上面 3 个平均速度算出来，写在一张表里。
动手算：\(\bar{v} = [x(2 + \Delta t) - x(2)] / \Delta t\)，\(x(t) = 5 t^2\)。
↗ 进阶推荐
如果你算完上面三个数字觉得"非常有趣"——比如发现它们好像在围绕 20 m/s 这个值打转——别急着记结论。这正是极限这一思想的力量。下一课《02_极限_物理引入》会从这种"打转"切入，把瞬时速度的严格定义建起来。

Card 4 · 加速度 · 描述"速度的变化有多快"

▼
学习目标：写出加速度的定义式；明确加速度的矢量性（方向规定）；区分"加速度大"与"速度大"——一个静止物体的加速度可以很大（如刚启动的火箭），一个高速物体的加速度可以为 0（匀速直线运动）。
建议时长：25-30 分钟
STEP 1先自己想
5 分钟

在草稿上写两件事的差别：

  1. "我跑得快" 与 "我跑得越来越快"——这两句话描述的是同一个物理量吗？
  2. 一辆车，匀速 100 km/h 与匀速 10 km/h，两辆的加速度分别是多少？
STEP 2加速度 a · 速度对时间的变化率
定义式 + 单位 + 方向
加速度（Acceleration · 矢量）的定义
文字定义：加速度是描述物体速度变化快慢（和方向）的物理量。"平均加速度"是这一定义在一段时间上的版本——用速度变化量除以所用时间。
翻译成数学语言：把"速度变化量"写成 \(\Delta \vec{v}\)（按矢量减法 \(v_{\text{终}} - v_{\text{始}}\)）、"时间"写成 \(\Delta t\)——上面这句中文就变成下面的定义式。
平均加速度的定义式：\(\bar{\vec{a}} = \dfrac{\Delta \vec{v}}{\Delta t}\)；在直线运动 + 规定正方向后：\(\bar{a} = \dfrac{\Delta v}{\Delta t} = \dfrac{v_{\text{终}} - v_{\text{始}}}{\Delta t}\)。
单位：m/s²（米每二次方秒），意为"每 1 秒，速度变化多少 m/s"。
矢量性：方向与 \(\Delta \vec{v}\) 同向。在直线运动中通过 \(\bar{a}\) 的正负号表达——\(\bar{a} > 0\) 表示加速度沿正方向；\(\bar{a} < 0\) 表示沿负方向。
关键认识 1：加速度描述的是速度变化的快慢，不是"运动的快慢"。匀速 100 km/h 的车，速度大但速度不变，所以加速度 = 0。
关键认识 2：在直线运动中，"加速度方向"和"速度方向"可能同向（加速）或反向（减速）。"加速度为负"不等于"减速"——要看 \(a\) 与 \(v\) 的正负号是同号还是异号。详见 Card 5。
思考点 7 · 能不能问"某一时刻的加速度"？
"平均加速度"算的是一段时间的速度变化。但能不能问"这一刻，速度变化得有多快"？类比 Card 3 STEP 3 瞬时速度的思路——\(\Delta t\) 越来越小时，\(\Delta v / \Delta t\) 会发生什么？
"不要直接给我'瞬时加速度的公式'。请引导我用 Card 3 STEP 3 同样的语言来描述：\(\Delta t\) 不断缩短，\(\Delta v / \Delta t\) 会不会也趋近一个固定的数？这个'趋近'背后的数学概念是什么？"
用一句话写下你的猜想：当 \(\Delta t\) 越来越小，\(\Delta v / \Delta t\) ____。
这道思考不要求现在写公式。下一课《02_极限_物理引入》会从这一类"越来越接近"的问题入手，把瞬时加速度的精确定义建起来。
思考点 8 · "加速度为负"一定是"减速"吗？
一辆车规定"向东为正"。某时刻 \(v = -5\) m/s（向西跑），\(a = -2\) m/s²。这辆车下一秒会"越跑越快"还是"越跑越慢"？方向有没有变？
"不要直接给我'加速 / 减速'的判断结论。请引导我对照'当 v 与 a 同号时是加速、异号时是减速'这条线索：v = −5、a = −2，两个负号是同号还是异号？所以下一秒速度的大小会怎么变？"
在草稿上写：v 的符号 = ____；a 的符号 = ____；同号 / 异号 = ____；下一秒速度大小 ____（变大 / 变小 / 不变）。
卡住的话：先想"如果 a 与 v 同号，物体被推往原来的方向，越推越快"——这是加速；反之"a 与 v 异号"是减速。

STEP 2v-t 图 · 斜率即加速度（图像版）
读图四步
v-t 图的"读法四步"
第一步 · 看纵轴正负：v > 0 = 物体沿正方向跑；v < 0 = 沿负方向跑；v = 0 = 静止。
第二步 · 看曲线斜率：斜率 = \(\Delta v / \Delta t\) = 平均加速度（这一段时间内的）。斜率 > 0 → a > 0；斜率 < 0 → a < 0；斜率 = 0（曲线水平）→ a = 0（匀速）。
第三步 · 合成判断"加速 / 减速"：v 与 a 同号 → 加速（速度大小变大）；v 与 a 异号 → 减速（速度大小变小）。这是 Card 4 思考点 8 的图像版本。
第四步 · 看曲线穿过 v = 0 的位置：当 v-t 曲线穿过横轴（v 从正变负或从负变正）时，物体在掉头——这一瞬间是运动方向反转的时间点。
同一段电梯运动（启动加速 5 秒 → 匀速 10 秒 → 减速 5 秒回到静止）的 v-t 图与 a-t 图对照。① 段 v-t 斜率为正（对应 a-t 中 a = +0.4 m/s²，位于横轴上方）；② 段斜率为 0（对应 a = 0，落在横轴上）；③ 段斜率为负（对应 a = −0.4 m/s²，落在横轴下方）。v-t 图的"斜率"与 a-t 图的"高度"一一对应。注：a-t 中两次"瞬时跳变"（虚线）是理想化——真实电梯加速度切换需要短暂过渡时间，这里把过渡简化为零，方便看清三段对应关系。
思考点 9 · 在 v-t 图上，加速度对应什么几何量？
回到加速度定义式 \(a = \Delta v / \Delta t\)。看 v-t 图——\(\Delta v\) 在图上是什么？\(\Delta t\) 在图上是什么？两者的比 \(\Delta v / \Delta t\) 对应几何上的什么？
"不要给我'斜率'两个字。请引导我用'纵向变化量 / 横向变化量'这条线索，自己得出 v-t 图上 \(\Delta v / \Delta t\) 对应的几何量是什么。"
在草稿上写：v-t 图的斜率对应物理量 = ____；当斜率 > 0 时 a ____；斜率 < 0 时 a ____；斜率 = 0 时 a ____。
↗ 进阶推荐
"v-t 图的斜率 = 加速度"在直觉上你已经接受了。下一课《02_极限_物理引入》会把"斜率"从平均（\(\Delta v / \Delta t\)）延伸到瞬时（\(dv/dt\)）；再下一课《03_幂函数微积分_直线运动公式》会把"v-t 图下方的面积"翻译成"位移"——这两件事一起构成"图像即定义"的完整版。本课只把"斜率"这一边立起来。
