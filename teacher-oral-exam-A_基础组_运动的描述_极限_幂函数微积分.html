选题器（5 选 3）

主问题 1

五概念束区分
哪些是矢量 · 哪些是标量

约 3 分钟
开放主问

描述一个物体的运动，我们需要的最少概念是哪几个？为什么仅凭「路程和速率」不够？请你举一个例子说明位移和路程会得到完全不同的数，并指出在这个例子里「加速度」是否一定为零。

追问树

深追 L1
  • 「参考系」和「质点」两个词，你刚才用到了吗？如果让你研究地球绕太阳的周期，可以把地球看成质点吗？研究地球上不同纬度白昼长短呢？
  • 同一个物体，两个不同参考系会描述出两种不同的运动——这件事在你刚才的例子里能不能也演示一下？
深追 L2
  • 把你的例子改成「往返一次回到起点」：位移、路程、平均速度、平均速率四个量分别等于多少？哪两个可能为 0？
  • 一辆车匀速 100 km/h，它的加速度是多少？现在给它一个 0.1 m/s2 的加速度，它的速度还是不是 100 km/h？「加速度大」和「速度大」是同一件事吗？
卡壳 · 台阶 1
  • 「先选一个参考系，再把研究对象抽象成一个点」——这两句话你认得吗？想想 01 课 Card 1 那张高铁里乘客喝水的图。
卡壳 · 台阶 2
  • 给你半个例子：你从教室门口走到讲台再走回门口。「实际走过的总长」是路程，「起点指向终点的有向线段」是位移——这趟来回，两个数各是多少？把它们写在纸上。

听点（学生说到位时话里应包含）

⭐参考系运动的描述必须先选参考系；同一物体在不同参考系下可呈现不同运动
⭐质点当物体大小相对研究问题可以忽略 / 或物体上各点运动状态相同时才能抽象为质点
⭐位移 vs 路程位移是矢量（起点→终点有向线段）/ 路程是标量（实际轨迹总长）；往返时位移可为 0 而路程不为 0
⭐速度 vs 速率平均速度 = 位移 / 时间 / 矢量；平均速率 = 路程 / 时间 / 标量
⭐加速度独立性加速度大小与速度大小无关；匀速运动 a = 0；静止物体也可有大 a（如刚启动的火箭瞬间）
⭐v-t 图直觉能口头描述「v-t 图斜率就是加速度」，但本主问不强求

误区信号

信号 A · 概念混用：把「位移」和「路程」两个词当同义词随手互换。 → 说明 矢量 / 标量 的根本区分没通；救场：「你刚才说的『走了 10 米』是位移还是路程？请用『起点指向终点』这个判据再说一遍。」
信号 B · 速度=加速度：说「匀速 100 km/h 加速度也大」或「静止加速度一定为 0」。 → 说明 a 的定义「速度的变化率」没通；救场：「a 描述的是速度怎么变，不是速度本身。一辆停在红灯前的车，绿灯起步那一瞬间速度还是 0，但 a 是 0 吗？」
信号 C · 背诵型：照本宣科「位移是矢量、路程是标量、平均速度是矢量、平均速率是标量」，问「为什么」立刻崩——指不出例子里哪个数对应哪个量。 → 说明 概念只停在术语层；救场：「你能不能不用『矢量』这个词重新讲一遍位移和路程的区别？」
信号 D · 忽略参考系：直接讲速度位移而完全不提「相对谁」。 → 说明 第一层「运动是相对的」没建立；救场：「你刚才说车开了 30 m/s，是相对谁说的？换成车上的乘客来看呢？」

救场话术（给台阶不给答案）

指路型 1：「01 课 Card 1 有一张高铁场景图，主语是『坐着喝水的乘客』，你回忆一下那张图想说明什么——再回到我们这道题。」
指路型 2：「你刚才举的例子是直线一次走 10 米。我把它改一下：走 10 米再走回原地。这时哪些数变、哪些数不变？」
指路型 3：「先别管『矢量』两个字。我换个问法——你最终人在哪里？走了多远的路？这两件事可以同时回答出来吗？」

主问题 2

平均速度为什么
答不了「此刻多快」

约 3 分钟
开放主问

一辆小车从 \(t = 0\) 到 \(t = 4\) s 走了 80 m，平均速度是 20 m/s。现在问你：在 \(t = 2\) s 这一瞬间，它的速度是多少？为什么平均速度的 20 m/s 不能直接当作答案？你打算怎么把「一段时间的平均」变成「某一时刻的瞬时」？

追问树

深追 L1
  • 你说「让 Δt 越来越小」——那能不能直接令 Δt = 0？这样做会得到什么表达式？这个表达式有意义吗？
  • 「无限接近」和「等于」是不是一回事？数学上用什么记号表示这种「无限接近」？
深追 L2
  • 假设这辆车的位置满足 \(x(t) = 5 t^2\)。在 \(t = 2\) s 的瞬时速度你能口头估计吗？告诉我你的步骤——不用算到底，把怎么算讲清楚就行。
  • 如果题目换成 \(x(t) = 5 t^2 + 3 t\)，你的步骤要不要改？哪一步会不同？
卡壳 · 台阶 1
  • 「平均」和「瞬时」两个词的区别——平均描述一段时间内的整体快慢；瞬时描述某一时刻的快慢。你能不能用你身边的例子（如自己跑步）举一个对比？
卡壳 · 台阶 2
  • 给你半个步骤：先算 \([2, 2.1]\) 这 0.1 秒内的平均速度，再算 \([2, 2.01]\) 这 0.01 秒内的平均速度，再算 \([2, 2.001]\) 的平均速度。把它们写出来你会看到什么趋势？

听点

⭐平均的局限平均速度只描述一段时间内的整体快慢 / 不描述某一瞬间
⭐数表法直觉取 Δt 越来越小的几组数值，观察 \(\Delta x / \Delta t\) 趋近某个确定值，这个值就是瞬时速度
⭐0/0 困惑解套不能令 Δt = 0（会得到无意义的 0/0），但能让 Δt「无限接近 0」——这正是极限的核心
⭐极限记号\(\displaystyle v(t) = \lim_{\Delta t \to 0} \dfrac{\Delta x}{\Delta t}\) · 能口头读出 / 不必书面默写整套
⭐不展开严格定义本课不要求 ε-δ 严格定义；学生若主动谈「无限接近≠等于」是亮点但不强求

误区信号

信号 A · 直接代 0：说「把 Δt 当 0 代进去 \(\Delta x / \Delta t\) 就得到瞬时速度」。 → 说明 极限「无限接近≠等于」没通；救场：「你把 Δt 代成 0，分子分母分别是什么？这个分式有意义吗？现在换『让 Δt 不断变小但不为 0』再说一次。」
信号 B · 用小段平均冒充瞬时：随手算一个 \([2, 2.1]\) 的平均速度就报作瞬时速度。 → 说明 没建立「取极限」这一步；救场：「这个 0.1 秒的平均，是瞬时吗，还是逼近瞬时的一个候选值？你还要做哪一步才能拿到瞬时？」
信号 C · 背诵型：能背「瞬时速度等于位移对时间的极限」但问「为什么不能令 Δt = 0」立刻崩。 → 说明 只记符号没通含义；救场：「抛开 lim 这个符号，你用『我让 Δt 越来越小』这种白话讲一遍这个过程。」
信号 D · 把 lim 当函数算：写出 \(\lim\) 后直接当函数操作，不解释它的含义。 → 说明 lim 只是被当作「一个套子」，不知其物理意义；救场：「这个 lim 在描述哪一个过程？指着它跟我讲一遍。」

救场话术

指路型 1：「02 课 Card 2 有一张 Δt 数表（0.1, 0.01, 0.001, …），让你看 \(\Delta x / \Delta t\) 的数值变化。回忆一下那张表想说明什么。」
指路型 2：「先别管 lim 符号。用大白话讲：什么叫『让一段时间不断缩短，但永远不变成 0』？」
指路型 3：「我给你一个具体函数 \(x = 5 t^2\)。请你不要直接套公式——从 \([2, 2.1]\) 和 \([2, 2.01]\) 两次平均速度开始算给我看。」

主问题 3

x-t 图上
割线如何变成切线

约 3 分钟
开放主问

你画一张 x-t 图（曲线随便画一条向上凸的）。在图上取两点 \(P_1\) 和 \(P_2\)，连成一条割线——这条割线的斜率在物理上等于什么？现在你让 \(P_2\) 沿曲线向 P1 逼近，这条割线会变成什么？它的斜率又等于什么？请你边在纸上画边讲。

追问树

深追 L1
  • 「割线斜率 = 平均速度」——这句话里的「平均」对应几何上的什么？为什么 \(P_2 \to P_1\) 之后，「平均」就变成了「瞬时」？
  • 切线只有一条吗？如果换 P1 的位置，切线会跟着变吗？
深追 L2
  • 同一个物体，如果它做匀速直线运动，x-t 图是什么样？这时割线和切线有什么关系？
  • 如果是匀加速直线运动呢？x-t 图是抛物线——切线斜率怎么变？这件事和 Q1 里「加速度方向」如何对应？
卡壳 · 台阶 1
  • 02 课 Card 3 那张「割线 → 切线」的几何图你还记得吗？两点连成的线叫什么？两点重合后剩下的那条切于曲线的线叫什么？
卡壳 · 台阶 2
  • 先回到代数：割线的斜率写成分式是 \(\dfrac{x_2 - x_1}{t_2 - t_1}\)。这个分式你认识吗？它对应物理上哪一个量？

听点

⭐割线斜率=平均速度x-t 图两点连成的割线，其斜率 \(\dfrac{x_2 - x_1}{t_2 - t_1} = \bar v\)
⭐切线斜率=瞬时速度当 \(P_2 \to P_1\) 时，割线的极限位置就是切线；切线斜率等于瞬时速度
⭐几何与代数对应能口头讲「逼近过程」在代数上对应 \(\lim_{\Delta t \to 0}\)，在几何上对应割线压向切线
⭐特殊情形匀速时 x-t 是直线 / 割线即切线 / 全程斜率相同（可主动提到，不强求）

误区信号

信号 A · 割线/切线混淆：把「两点连线」叫切线，或者把「过一点的某条直线」叫割线。 → 说明 几何术语未与曲线行为挂钩；救场：「割线交曲线于 2 点；切线在该点『贴住』曲线（极限位置）。你刚才画的那条线，它和曲线交了几次？」
信号 B · 反向类推 v-t：把「x-t 图斜率 = 速度」错说成「v-t 图斜率 = 速度」。 → 说明 两种图像区分未通；救场：「x-t 图纵轴是位移，所以斜率是 Δx/Δt——速度。v-t 图纵轴是速度，斜率是 Δv/Δt——那是什么？」
信号 C · 背诵型：能背「割线 → 切线 → 瞬时速度」但拿张图让他指出割线 / 切线立刻指错。 → 说明 概念未与图像绑定；救场：「请你边画边讲，不准用『割线』『切线』这两个词，用自己的话描述这个过程。」
信号 D · 只讲代数不讲几何：能写出 \(\dfrac{x_2 - x_1}{t_2 - t_1}\) 但被问「在图上指出这条线」立刻卡。 → 说明 代数与几何两层未打通；救场：「你现在在草稿上画两个坐标点 P1、P2——用尺连起来，这条实线就是你写的那个分式的几何形象。」

救场话术

指路型 1：「02 课 Card 3 的几何图——你能从书包里翻出当时的草稿吗？指着图讲。」
指路型 2：「先把割线的代数表达式 \(\dfrac{x_2 - x_1}{t_2 - t_1}\) 写出来。这是物理上的什么？现在让 \(t_2\) 越来越接近 \(t_1\)——分式会变成什么样？」

主问题 4

幂函数求导
从极限定义推 (xn)′

约 3 分钟
开放主问

请你不要直接报结论。从极限定义出发，推一遍 \((x^2)' = 2x\)——讲思路就行，关键代数步骤口头说，纸上写 1–2 行最关键的展开式。推完后告诉我：这个结论怎么从 \(n = 2\) 推广到一般的 \(x^n\)？

追问树

深追 L1
  • 你展开 \((x + \Delta x)^2 = x^2 + 2 x \Delta x + (\Delta x)^2\)——为什么最后只剩 \(2x\)？\((\Delta x)^2\) 那一项哪去了？
  • 把 \(n = 2\) 换成 \(n = 3\) 再来一次，你的步骤会变吗？关键代数动作是什么？
深追 L2
  • 反过来：如果给你 \(\int x^n \, dx = \dfrac{x^{n+1}}{n+1} + C\)（\(n \neq -1\)），你能用「反求导直觉」解释这条公式吗？为什么要排除 \(n = -1\)？
  • 取一个特例 \(n = 0\)（即 \(x^0 = 1\)）。\(\int 1 \, dx = ?\)，和你公式得到的结果对得上吗？
卡壳 · 台阶 1
  • 导数的定义是什么？写出 \(f'(x) = \lim_{\Delta x \to 0} \dfrac{f(x + \Delta x) - f(x)}{\Delta x}\) 这个式子，然后让 \(f(x) = x^2\) 代进去看看。
卡壳 · 台阶 2
  • 展开 \((x + \Delta x)^2\) 后，分子是 \(2 x \Delta x + (\Delta x)^2\)。这个分子的每一项除以分母 \(\Delta x\) 各是什么？现在让 \(\Delta x \to 0\)——哪一项留下、哪一项消失？

听点

⭐导数定义能口头说出 \(f'(x) = \lim_{\Delta x \to 0} \dfrac{f(x + \Delta x) - f(x)}{\Delta x}\) 的物理与几何含义
⭐代数展开能口头讲清 \((x + \Delta x)^2 = x^2 + 2 x \Delta x + (\Delta x)^2\) 的展开 / 抵消 \(x^2\) / 除 \(\Delta x\) / 取极限四步
⭐(xn)′ 结论(xn)′ = n xn−1 · 至少对 n = 2 能现场重推
⭐∫ xn dx 反求导能用「求导的反过程」解释 \(\int x^n dx = \dfrac{x^{n+1}}{n+1} + C\)；知道 n = −1 要排除
⭐二项式直觉能口头点出「高阶项 \((\Delta x)^k\) 在 Δx→0 时被吃掉」 / 不强求

误区信号

信号 A · 跳过推导直接报结论：被问「推一遍」立刻报 \((x^n)' = n x^{n-1}\)，没有过程。 → 说明 公式只在记忆层；救场：「我不是要你告诉我答案。请你从极限定义那一行开始写——不展开就讲不清。」
信号 B · (Δx)2 项处理错：保留 \((\Delta x)^2 / \Delta x = \Delta x\) 这一项作为最终答案的一部分。 → 说明 极限过程未到位；救场：「你写完 \(2x + \Delta x\) 之后，要做的最后一步是什么？让 Δx 怎么样？」
信号 C · 反求导与求导混淆：把 \(\int x^n dx\) 直接写成 \(n x^{n-1}\) 或类似形式。 → 说明 求导 / 积分两个动作没区分；救场：「积分是求导的反过程。\(\dfrac{d}{dx}\bigl(\dfrac{x^{n+1}}{n+1}\bigr) = ?\) 你算一下，能不能得回 \(x^n\)？」
信号 D · 背诵型：能背公式但要求「用自己的话解释为什么不能 n = −1」立刻卡。 → 说明 公式适用条件未理解；救场：「把 n = −1 代到 \(\dfrac{x^{n+1}}{n+1}\) 里看看分母是几？这个表达式还有意义吗？」

救场话术

指路型 1：「03 课 Card 1 STEP 3 让你亲手推过 (x2)′——回忆当时你写的展开式是哪 3 行。」
指路型 2：「先写定义那一行：\(f'(x) = \lim_{\Delta x \to 0} \dots\)。把 \(f(x) = x^2\) 代进去——分子能展开成什么样？」
指路型 3：「反求导只问一件事：哪个函数对 x 求导后得到 \(x^n\)？你猜一个，再求导验证。」

主问题 5

三公式独立推导
v = v0 + at · x = v0t + ½at2 · v2 = v02 + 2ax

约 3 分钟
开放主问

请你从加速度的定义 \(a = \dfrac{dv}{dt}\) 出发（\(a\) 为常数），不用记忆中的成品，独立推导匀加速直线运动的三个公式：先推 \(v\)，再推 \(x\)，最后推不显含 \(t\) 的公式。每一步告诉我「为什么能这么写」。

追问树

深追 L1
  • 推 \(x = v_0 t + \dfrac{1}{2} a t^2\) 时，那个 \(\dfrac{1}{2}\) 是怎么来的？它对应 v-t 图上的什么几何量？
  • 第三公式 \(v^2 = v_0^2 + 2 a x\) 你用的是代数消 \(t\)，还是 \(a \, dx = v \, dv\) 直接积分？两条路径都试试。
深追 L2
  • 当 \(a\) 不是常数（如弹簧、阻力）时，三公式还能用吗？你的推导哪一步会出问题？
  • 自由下落 / 刹车这类问题里，「正方向」怎么定？\(a\) 与 \(v_0\) 的正负号会变吗？
卡壳 · 台阶 1
  • 加速度的定义 \(a = \dfrac{dv}{dt}\) 你认得吧。把它两边对时间积分——左边是什么？右边是什么？
卡壳 · 台阶 2
  • 给你半个推导：\(\int_0^t a \, dt' = a t\)（因为 a 是常数），所以 \(v(t) - v_0 = a t\) → \(v = v_0 + a t\)。下一步你想用 \(v = \dfrac{dx}{dt}\) 还是 v-t 图的面积？

听点

⭐v 的推导由 \(a = dv/dt\) 对 \(t\) 积分 / \(a\) 是常数提出 / 得 \(v = v_0 + a t\)
⭐x 的推导由 \(v = dx/dt\) 代入上式再对 \(t\) 积分 / 得 \(x = v_0 t + \dfrac{1}{2} a t^2\) / 能口头解释 ½ 的几何来源（v-t 图三角形面积）
⭐v² 公式两路径能讲清「代数消 t」或「\(a \, dx = v \, dv\) 直接积分」两条路径之一（任一即可）
⭐适用前提三公式仅对 \(a\) = 常数（匀加速直线运动）成立；变力 / 变 a 不再适用
⭐正方向与符号一维问题先定正方向 / \(v_0, v, a, x\) 按矢量代正负 / 刹车时 \(a < 0\)（可主动谈到，不强求）

误区信号

信号 A · 默写式套用：直接报「v 等于 v0 加 at」再「x 等于 v0t 加二分之一 at 平方」，不讲来源。 → 说明 公式停在记忆层；救场：「我不是问公式长什么样。请你从 \(a = dv/dt\) 这一行开始——下一步是积分还是别的？」
信号 B · ½ 来源说不清：能写出 \(x = v_0 t + \dfrac{1}{2} a t^2\)，但被问「½ 是怎么出来的」立刻卡。 → 说明 积分过程未消化；救场：「\(\int_0^t (v_0 + a t') \, dt' = v_0 t + ?\)——把第二项积出来你看。或者画 v-t 图：v 从 v0 直线增长到 v0+at，下面那个梯形面积怎么算？」
信号 C · 适用条件忽略：被问「a 是变化的怎么办」答「也能用」。 → 说明 推导前提没意识；救场：「你刚才在哪一步用了『a 是常数』？把 a 换成 a(t) 看看那一步还能不能做。」
信号 D · 三公式当三件独立东西背：能背出三公式但说不出它们之间是「由 a 一步步积出来」的推导链。 → 说明 链路断开 / 背诵型；救场：「这三个公式不是三件事。第一个推第二个，第一个第二个共推第三个——画一张你脑中的推导路线图给我看。」

救场话术

指路型 1：「03 课 Card 3、Card 4、Card 5 各推一公式——回忆 Card 3 那张『从 a 积到 v』的两行式子。」
指路型 2：「先把『\(a\) 是常数』四个字写在纸上当条件。然后写 \(a = dv/dt\)。两边对 \(t\) 积分——左边是什么，右边是什么？」
指路型 3：「画一张 v-t 图：横轴 t、纵轴 v；从 v0 起，斜率 a 的直线。你画完后，『位移』对应图上哪个量？面积怎么算？」
