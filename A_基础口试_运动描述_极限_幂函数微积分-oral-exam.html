关于深度口试

口试是 运动的描述, 极限（物理引入）, 幂函数微积分与直线运动公式 的真实通关环节：先把学习任务卡学完、基础检测练熟，再预约口试。本页是 完整题库，公开给所有学生 —— 你可以提前准备每一条主问题。

口试由教师当面进行，从下方主问题中抽 3 条。每条主问题先开放讲，再由教师根据你的回答追问。讲思路比报出唯一正确数值更重要 —— 教师听的是因果重建。

📌 通关条件：抽中的 3 条里，至少 2 条把核心要点讲清楚并能回应一层「为什么」，且联动环节能在错题或推导手稿上现场指出一处「当时怎么想」。

口试流程 总时长 12 分钟内

① 暖场 · 自述0-2 分钟你先讲：「这几课里，让你印象最深的一句结论是什么？为什么重要？」—— 不卡时间，讲到能切入主问题就行。
② 主问题 ×32-10 分钟从下方主问题中由教师抽 3 条，每条约 3 分钟左右。先开放讲，再回答追问。
③ 联动 · 收尾10-11 分钟带 1-2 道自己曾经做错的题或一份推导手稿，现场讲「当时为什么错、现在怎么想」。
④ 判定 · 签字11-12 分钟通过 = 当场签字；暂缓 = 给补学方向 + 预约二试（二试必须换未用过的主问题）；不通过 = 回到学习阶段重学。
主问题 1

五概念束区分 · 哪些是矢量 / 哪些是标量

约 3 分钟
🎤 开放主问：描述一个物体的运动，我们需要的最少概念是哪几个？为什么仅凭「路程和速率」不够？请你举一个例子说明位移和路程会得到完全不同的数，并指出在这个例子里「加速度」是否一定为零。
🎯 教师会重点听这些考查锚点
  • ⭐⭐参考系 运动的描述必须先选参考系；同一物体在不同参考系下可呈现不同运动。
  • ⭐⭐质点抽象 物体大小相对研究问题可以忽略 / 或物体上各点运动状态相同时才能抽象为质点。
  • ⭐⭐位移 vs 路程 位移是矢量（起点→终点有向线段）/ 路程是标量（实际轨迹总长）；往返时位移可为 0 而路程不为 0。
  • ⭐⭐速度 vs 速率 平均速度 = 位移 / 时间（矢量）；平均速率 = 路程 / 时间（标量）。
  • ⭐⭐加速度独立性 加速度大小与速度大小无关；匀速运动 a = 0；静止物体也可有大 a（如刚启动的火箭瞬间）。
  • ⭐v-t 图直觉 能口头说出「v-t 图斜率就是加速度」是亮点；主动谈到不展开也行。
主问题 2

平均速度为什么答不了「此刻多快」

约 3 分钟
🎤 开放主问：一辆小车从 \(t = 0\) 到 \(t = 4\) s 走了 80 m，平均速度是 20 m/s。现在问你：在 \(t = 2\) s 这一瞬间，它的速度是多少？为什么平均速度的 20 m/s 不能直接当作答案？你打算怎么把「一段时间的平均」变成「某一时刻的瞬时」？
🎯 教师会重点听这些考查锚点
  • ⭐⭐平均的局限 平均速度只描述一段时间内的整体快慢，不描述某一瞬间。
  • ⭐⭐数表法直觉 取 Δt 越来越小的几组数值，观察 \(\Delta x / \Delta t\) 趋近某个确定值，这个值就是瞬时速度。
  • ⭐⭐0/0 困惑解套 不能令 Δt = 0（会得到无意义的 0/0），但能让 Δt「无限接近 0」——这正是极限的核心。
  • ⭐⭐极限记号 能口头读出 \(\displaystyle v(t) = \lim_{\Delta t \to 0} \dfrac{\Delta x}{\Delta t}\) 的含义。
  • ⭐不展开严格定义 本组不要求 ε-δ 严格定义；主动谈「无限接近≠等于」是亮点。
主问题 3

x-t 图上 · 割线如何变成切线

约 3 分钟
🎤 开放主问：你画一张 x-t 图（曲线随便画一条向上凸的）。在图上取两点 \(P_1\) 和 \(P_2\)，连成一条割线——这条割线的斜率在物理上等于什么？现在你让 \(P_2\) 沿曲线向 P1 逼近，这条割线会变成什么？它的斜率又等于什么？请你边在纸上画边讲。
🎯 教师会重点听这些考查锚点
  • ⭐⭐割线斜率 = 平均速度 x-t 图两点连成的割线，其斜率 \(\dfrac{x_2 - x_1}{t_2 - t_1} = \bar v\)。
  • ⭐⭐切线斜率 = 瞬时速度 当 \(P_2 \to P_1\) 时，割线的极限位置就是切线；切线斜率等于瞬时速度。
  • ⭐⭐几何与代数对应 能口头讲「逼近过程」在代数上对应 \(\lim_{\Delta t \to 0}\)，在几何上对应割线压向切线。
  • ⭐特殊情形 匀速时 x-t 是直线 / 割线即切线 / 全程斜率相同；主动谈到是亮点。
主问题 4

幂函数求导 · 从极限定义推 (xn)′

约 3 分钟
🎤 开放主问：请你不要直接报结论。从极限定义出发，推一遍 \((x^2)' = 2x\)——讲思路就行，关键代数步骤口头说，纸上写 1-2 行最关键的展开式。推完后告诉我：这个结论怎么从 \(n = 2\) 推广到一般的 \(x^n\)？
🎯 教师会重点听这些考查锚点
  • ⭐⭐导数定义 能口头说出 \(f'(x) = \lim_{\Delta x \to 0} \dfrac{f(x + \Delta x) - f(x)}{\Delta x}\) 的物理与几何含义。
  • ⭐⭐代数四步 能口头讲清 \((x + \Delta x)^2 = x^2 + 2 x \Delta x + (\Delta x)^2\) 的展开 / 抵消 \(x^2\) / 除 \(\Delta x\) / 取极限四步。
  • ⭐⭐(xn)′ 结论 (xn)′ = n xn−1；至少对 n = 2 能现场重推。
  • ⭐⭐∫ xn dx 反求导 能用「求导的反过程」解释 \(\int x^n dx = \dfrac{x^{n+1}}{n+1} + C\)；知道 n = −1 要排除。
  • ⭐二项式直觉 能口头点出「高阶项 \((\Delta x)^k\) 在 Δx→0 时被吃掉」是亮点。
主问题 5

三公式独立推导 · v = v0 + at / x = v0t + ½at2 / v2 = v02 + 2ax

约 3-4 分钟
🎤 开放主问：请你从加速度的定义 \(a = \dfrac{dv}{dt}\) 出发（\(a\) 为常数），不用记忆中的成品，独立推导匀加速直线运动的三个公式：先推 \(v\)，再推 \(x\)，最后推不显含 \(t\) 的公式。每一步告诉我「为什么能这么写」。
🎯 教师会重点听这些考查锚点
  • ⭐⭐v 的推导 由 \(a = dv/dt\) 对 \(t\) 积分；\(a\) 是常数提出；得 \(v = v_0 + a t\)。
  • ⭐⭐x 的推导 由 \(v = dx/dt\) 代入上式再对 \(t\) 积分；得 \(x = v_0 t + \dfrac{1}{2} a t^2\)；能口头解释 ½ 的几何来源（v-t 图三角形面积）。
  • ⭐⭐v² 公式两路径 能讲清「代数消 t」 或 「\(a \, dx = v \, dv\) 直接积分」两条路径之一（任一即可）。
  • ⭐⭐适用前提 三公式仅对 \(a\) = 常数（匀加速直线运动）成立；变力 / 变 a 不再适用。
  • ⭐正方向与符号 一维问题先定正方向；\(v_0, v, a, x\) 按矢量代正负；刹车时 \(a < 0\)。主动谈到是亮点。

怎么准备

  • 不要背口诀。教师追问的第一刀往往就是「为什么」—— 把每个核心结论的来源 + 一句话理由想清楚（例如：瞬时速度的 0/0 困惑不是真分母为 0，而是「让 Δt 无限接近 0 时 Δx/Δt 趋近的那个确定值」）。
  • 带纸和笔。主问题 2、3、4、5 都涉及在纸上写公式或画图：x-t 图割线→切线、(x²)′ 的代数四步、三公式的独立推导。
  • 提前挑 1-2 道自己错过的题（基础检测里的、或学习任务卡 checkpoint 失分项），准备讲「当时为什么错、现在怎么想」。这是流程第 ③ 步。
  • 边界标 ⭐（银）的不强求：v-t 图直觉、ε-δ 严格定义、x-t 直线特殊情形、二项式直觉、刹车 \(a<0\) 符号约定 —— 主动谈到是亮点，没提到也不扣分。
  • 讲思路 ＞ 算对答案。教师听的是因果重建：你能不能把「为什么这样、不那样」讲清楚，比报出唯一正确数值更重要。
