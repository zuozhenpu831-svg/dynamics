本课定位：直线运动公式的"出生证"
本课是直线运动板块的数学工具课，但更是三公式的诞生现场。前两课你已经能用极限定义瞬时速度，本课要把"极限"升级为导数与定积分这两件可重复使用的工具，并用它们把匀变速直线运动的三条公式（v = v₀ + at / x = v₀t + ½at² / v² = v₀² + 2ax）从加速度定义里一行一行推出来。三公式不是要背的"结论"，是要会"推"的工具。

学习目标

  1. 掌握幂函数求导：用极限定义直接推出 \((x^n)' = n x^{n-1}\)，至少能从 \(n=2\) 起步亲手推一遍
  2. 理解定积分的面积语义：\(\int_a^b f(x)\,dx\) 就是函数曲线下从 \(a\) 到 \(b\) 的面积，由"割条求和→极限"得到
  3. 掌握幂函数的不定积分：\(\int x^n\,dx = \dfrac{x^{n+1}}{n+1} + C\)（\(n \neq -1\)），通过反求导直觉接受这一公式
  4. 独立推导 v = v₀ + at：由加速度定义 \(a = \dfrac{dv}{dt}\) 对时间积分得到
  5. 独立推导 x = v₀t + ½at²：由速度定义 \(v = \dfrac{dx}{dt}\) 与上一公式联立积分得到
  6. 独立推导 v² = v₀² + 2ax：由代数消去 t、或由 \(a\,dx = v\,dv\) 直接积分两条路径完成
  7. 能解释三公式各自的物理含义、适用条件（匀加速直线运动）和几何来源（v-t 图面积）
驱动问题

第 1 课你写下了加速度的定义 \(a = \dfrac{\Delta v}{\Delta t}\)；第 2 课你接受了"\(\Delta t \to 0\) 时比值会趋近一个稳定数"。现在反过来问：如果加速度 \(a\) 是一个常数（匀变速直线运动），那么速度 \(v(t)\) 是关于时间的什么函数？位移 \(x(t)\) 又是什么？仅靠"定义 + 直觉"答不出来——需要一对新的数学工具。

本课的目标：用幂函数的导数与定积分这对新工具，把"加速度 = 速度对时间的变化率"这一句直觉反向展开，推出 \(v(t)\) 与 \(x(t)\) 的精确表达式——也就是匀变速直线运动的两条基础公式 \(v = v_0 + at\) 与 \(x = v_0 t + \dfrac{1}{2} a t^2\)；第三条 \(v^2 = v_0^2 + 2 a x\) 会作为推论自然得到。本课结束时，你不是"记"三公式，而是"推"三公式。

（剧透一句：\(\dfrac{1}{2}\) 这个系数不是记忆，而会自然冒出来——它来自 v-t 图里的一个三角形面积，而三角形面积你已经会算。本课就是把"三角形面积 = 位移"这件事讲清楚，并把它推广到任意常数 \(a\)。）

STEP 0 · 本课骨架（3 分钟读完）

读题前先把全图刻进脑子，免得做卡 1 时迷失在公式细节里。

骨架 1 · 两个新工具：导数与定积分
导数是"瞬时变化率"——前一课的瞬时速度 \(v = \lim\limits_{\Delta t \to 0} \dfrac{\Delta x}{\Delta t}\) 改个名字就是 \(v = \dfrac{dx}{dt}\)。同理 \(a = \dfrac{dv}{dt}\)。
定积分是"累积量"——把 v-t 图按时间割成无数细条，每条面积 \(v(t)\,dt\) 是一小段位移，所有细条面积加起来就是总位移：\(x = \int_0^t v\,dt\)。
本课只学幂函数 \(x^n\) 这一类的导数与定积分（线性运动里出现的都是它）。其他函数（三角、指数）留给后续。
骨架 2 · 两件关键公式（要"推"不要"背"）
幂函数求导：\((x^n)' = n x^{n-1}\)。卡 1 用极限定义亲手推。
幂函数不定积分：\(\int x^n\,dx = \dfrac{x^{n+1}}{n+1} + C\)（\(n \neq -1\)）。卡 2 用反求导直觉理解 + 面积法验证。
注意："求导→求一阶变化率"是"降幂"（\(t^2 \to 2t\)），"积分→求累积"是"升幂"（\(t \to \tfrac{1}{2} t^2\)）。两步刚好互逆——这正是后面要学的牛顿-莱布尼茨公式的直觉，下一课正式证明，本课只当工具用。
骨架 3 · 三公式的"出生路线图"
起点是一条事实：匀加速直线运动里，加速度 a 是常数。
路线 A（卡 3）：\(a = \dfrac{dv}{dt}\) 对时间积分 → \(v = v_0 + at\)
路线 B（卡 4）：\(v = \dfrac{dx}{dt}\) 代入路线 A 再积分 → \(x = v_0 t + \tfrac{1}{2} a t^2\)
路线 C（卡 5）：消去 t（代数法）或 \(a\,dx = v\,dv\) 直接积分 → \(v^2 = v_0^2 + 2 a x\)
读完合上页面，在草稿纸上把这条路线图画出来——画得出，再往下进卡 1；画不出，回到本段再读一遍。

卡 1 · 幂函数求导：用极限定义直接推 (xn)′ = n·xn-1

▼
学习目标：从极限定义出发，亲手推出 \((x^n)' = n x^{n-1}\)，并把它运用到 \(x^2, x^3\) 等具体情形。
建议时长：50 分钟（前置回顾 5 + 自推 20 + 对照 + 推广 25）
STEP 1前置回顾
5 分钟

来自第 2 课 · 极限：

当 \(\Delta t \to 0\)（或 \(\Delta x \to 0\)）时，分母变量永远 不等于 0、只是趋近 0，所以可以先约分再代值。例如： \[ \lim_{\Delta x \to 0} \frac{2x\,\Delta x + \Delta x^2}{\Delta x} = \lim_{\Delta x \to 0} (2x + \Delta x) = 2x. \]

本卡用到的全部极限技巧只有"先约分，再代 0"这一条。

来自第 1 课 · 运动的描述：

瞬时速度 \(v(t) = \lim\limits_{\Delta t \to 0} \dfrac{x(t+\Delta t) - x(t)}{\Delta t}\)。本卡把 \(x(t)\) 换成抽象的 \(f(x)\)，把时间换成自变量 \(x\)，结构完全相同——这就是"导数"。

STEP 2导数的定义
把瞬时速度推广

对任意函数 \(f(x)\)，它在点 \(x\) 处的导数定义为：

导数 · 极限形式定义
\[ f'(x) = \lim_{\Delta x \to 0} \frac{f(x + \Delta x) - f(x)}{\Delta x}. \]
读作 "f prime of x"；也可以写成 \(\dfrac{df}{dx}\)（莱布尼茨记号）。
物理意义：\(f'(x)\) 是 \(f\) 关于 \(x\) 的瞬时变化率。若 \(f\) 是位移、\(x\) 是时间，则 \(f' = v\)；若 \(f\) 是速度、\(x\) 是时间，则 \(f' = a\)。
思考点 1 · 把"瞬时速度"翻译成"导数"
把第 1 课的瞬时速度公式 \(v(t) = \lim\limits_{\Delta t \to 0} \dfrac{x(t+\Delta t) - x(t)}{\Delta t}\) 和上面的导数定义对照看——它们除了字母不同，结构是否完全一样？
我看到 v(t) 的极限定义和 f'(x) 的极限定义结构一致，但不太确定两者是不是同一件事。请你不要直接给答案，反过来问我：你能不能指出两个式子里"被求差的量"和"分母的变量"各自是谁？
用一句话写出："瞬时速度就是位移函数 \(x(t)\) 对时间 \(t\) 的______。"
填空答案是"导数"。如果你能写出来，就完全打通了第 1-2 课与本课。
STEP 3动手推 (x²)′
15 分钟

合上页面，在草稿上独立完成下面的推导。目标：导出 \((x^2)' = 2x\)。

思考点 2 · 亲手推 (x²)′
先写出 \(f(x) = x^2\) 时的导数定义；下一步把 \((x + \Delta x)^2\) 展开；再化简分子，最后令 \(\Delta x \to 0\)。每一步在草稿上单独写一行。
我推到第 ___ 步卡住了。请不要给完整答案，只指出我的式子是不是已经可以约分；以及"约掉的那个量在分母里是不是真的非零"。
把你的 4 行推导抄到正式纸上：① 套定义；② 展开平方；③ 化简（注意约分理由）；④ 代极限。
展开公式：\((x + \Delta x)^2 = x^2 + 2x\,\Delta x + (\Delta x)^2\)。
对照答案 · (x²)′ 的极限推导
① 套定义：\(\displaystyle (x^2)' = \lim_{\Delta x \to 0} \frac{(x+\Delta x)^2 - x^2}{\Delta x}\)
② 展开平方：\(\displaystyle = \lim_{\Delta x \to 0} \frac{x^2 + 2x\,\Delta x + (\Delta x)^2 - x^2}{\Delta x} = \lim_{\Delta x \to 0} \frac{2x\,\Delta x + (\Delta x)^2}{\Delta x}\)
③ 化简：因为 \(\Delta x \to 0\) 但 \(\Delta x \ne 0\)，可以把分子分母的 \(\Delta x\) 一起约掉：\(\displaystyle = \lim_{\Delta x \to 0} (2x + \Delta x)\)
④ 代极限：\(\displaystyle = 2x + 0 = 2x\)
结论：\((x^2)' = 2x\)。
STEP 4推广到 xn
用二项式直觉

从 \((x^2)' = 2x\) 出发，下一步是看 \((x^3)' = ?\)，并由此推广到一般的 \(n\)。

(x³)′ 推导（按相同方法）
套定义：\(\displaystyle (x^3)' = \lim_{\Delta x \to 0} \frac{(x+\Delta x)^3 - x^3}{\Delta x}\)
展开立方：\((x+\Delta x)^3 = x^3 + 3 x^2 \Delta x + 3 x (\Delta x)^2 + (\Delta x)^3\)
分子去掉 \(x^3\)、与分母约 \(\Delta x\)：\(\displaystyle = \lim_{\Delta x \to 0} \left[ 3 x^2 + 3 x \Delta x + (\Delta x)^2 \right]\)
代极限：\(\displaystyle = 3 x^2\)。
思考点 3 · 从具体 n 到一般 n · 下一步该做什么
现在 \((x^2)' = 2x\) 和 \((x^3)' = 3x^2\) 都用极限定义推出来了。但要说"对任意整数 \(n\) 都成立"，一个一个验证永远验证不完。下一步该做什么？是继续代 \(n=4, 5, 6\)，还是找一种能一次覆盖所有 \(n\) 的方法？
"不要直接告诉我答案。请引导我探究：我推 \((x^3)'\) 时用到了 \((x+\Delta x)^3 = x^3 + 3x^2\Delta x + 3x(\Delta x)^2 + (\Delta x)^3\) 这个展开。如果 \(n\) 是任意整数，\((x+\Delta x)^n\) 也能展开吗？展开后有没有可能有类似的结构——一个不含 \(\Delta x\) 的首项（会消去）+ 一个含 \(\Delta x\) 但不含更高次的线性项 + 一堆含 \((\Delta x)^{\geq 2}\) 的高阶项？"
卡住的话：回顾中学学过的"二项式定理" \((a+b)^n = \sum \binom{n}{k} a^{n-k} b^k\)。把 \(a=x, b=\Delta x\) 代进去，你能不能只写出前两项就够判断结论？
在草稿上写下"覆盖所有 \(n\) 的策略"：________（提示：不是"猜"，是"用一个能容纳所有 \(n\) 的展开工具"）。
用二项式定理推导 \((x^n)'\) · 每一步该做什么 + 为什么这么做
目标：对任意正整数 \(n\)，推出 \((x^n)'\) 的严格表达式。唯一严格出发点是导数的极限定义（这一点在极限一课里立起来了，此处直接调用）。
第 1 步 · 套定义：把 \(f(x) = x^n\) 代入导数定义式，得 \[ (x^n)' \;=\; \lim_{\Delta x \to 0} \frac{(x+\Delta x)^n - x^n}{\Delta x}. \] 为什么这么做：从"导数是什么"出发，一切幂函数求导都必须先经过这一步——这是不可绕过的定义关。
第 2 步 · 展开 \((x+\Delta x)^n\)：用二项式定理 \[ (x+\Delta x)^n \;=\; x^n \;+\; \binom{n}{1}\,x^{n-1}\,\Delta x \;+\; \binom{n}{2}\,x^{n-2}\,(\Delta x)^2 \;+\;\cdots\;+\;(\Delta x)^n. \] 利用 \(\binom{n}{1} = n\)，改写为 \[ (x+\Delta x)^n \;=\; x^n \;+\; n\,x^{n-1}\,\Delta x \;+\;\underbrace{\binom{n}{2}x^{n-2}(\Delta x)^2 + \cdots + (\Delta x)^n}_{\text{每项都含 }(\Delta x)^{\geq 2}}. \] 为什么这么做：分子里的 \((x+\Delta x)^n - x^n\) 是"未知的一大坨"。展开的目的是让 \(x^n\) 出现在展开式首项——为下一步与分子里的 \(-x^n\) 相消做准备。
第 3 步 · 分子相消 \(x^n\)：代入分子 \[ (x+\Delta x)^n - x^n \;=\; n\,x^{n-1}\,\Delta x \;+\; \binom{n}{2}x^{n-2}(\Delta x)^2 + \cdots + (\Delta x)^n. \] 为什么这么做：\(x^n\) 是分子里的"不含 \(\Delta x\) 的死项"。不消掉它，分母 \(\Delta x \to 0\) 时分子不趋于 0——极限根本不存在。这一步是让"\(\frac{0}{0}\) 型式子"露出可计算的结构。
第 4 步 · 分子分母同除 \(\Delta x\)：每一项都含 \(\Delta x\) 因子，同除得 \[ \frac{(x+\Delta x)^n - x^n}{\Delta x} \;=\; n\,x^{n-1} \;+\; \binom{n}{2}x^{n-2}\,\Delta x \;+\;\cdots\;+\;(\Delta x)^{n-1}. \] 为什么这么做：极限过程是 \(\Delta x \to 0\) 但 \(\Delta x \neq 0\)——所以可以约分。这是极限定义里"趋近但不到达"这一特殊规则允许我们做的事，中学代数里 \(\frac{0}{0}\) 不允许，但极限允许。
第 5 步 · 取极限：让 \(\Delta x \to 0\)。等式右边有一项 \(n\,x^{n-1}\) 不含 \(\Delta x\)——它保持不动；其余每一项都含 \(\Delta x^k\)（\(k \geq 1\)），全部趋于 0。因此 \[ (x^n)' \;=\; \lim_{\Delta x \to 0} \left[ n\,x^{n-1} \;+\; \binom{n}{2}x^{n-2}\,\Delta x \;+\;\cdots\;+\;(\Delta x)^{n-1}\right] \;=\; n\,x^{n-1}. \] 为什么这么做：这一步不是"忽略高阶项"——是严格的极限运算：任何含 \(\Delta x\) 因子的项在 \(\Delta x \to 0\) 时都精确地趋于 0，不是近似为 0。
思考点 3.5 · 为什么恰好是 \(n\,x^{n-1}\) 这一项"幸存"
在第 5 步里，为什么只有 \(n\,x^{n-1}\) 这一项在取极限后不消失？它有什么"结构上的特殊性"？
"不要直接告诉我答案。请引导我探究：在二项式展开里，我关心哪一项在'除以 \(\Delta x\) + 取极限' 后能留下不为 0 的东西。这一项在展开中含 \(\Delta x\) 的次数是多少？除以 \(\Delta x\) 后剩下几次 \(\Delta x\)？取极限后为什么它是唯一保留下来的？"
对比"高阶项 \(\binom{n}{2} x^{n-2}(\Delta x)^2\)"——它含 \((\Delta x)^2\)，除以 \(\Delta x\) 后剩 \(\Delta x\)，取极限归零。唯一在展开里含 \(\Delta x\) 恰好一次的项——它除以 \(\Delta x\) 后不含 \(\Delta x\)、系数是 \(\binom{n}{1} = n\)、幂是 \(x^{n-1}\)——所以只有它能"幸存"。
幂函数求导的通用结论（可迁移使用）
\[ \boxed{\,(x^n)' = n\, x^{n-1}\,} \]
这是一条通用结论：由二项式展开 + 极限定义严格推得，不是"猜到的规律"。对所有整数 \(n\) 都成立（对全体实数 \(n\) 也成立，本课只用整数 \(n\)）。
可迁移的使用场景：本课下面的匀变速三大公式（\(v\)、\(x\)、\(v^2\)）、卡 3 加速度对时间求导、卡 4 变量替换后的求导，都会直接调用这一结论——不需要每次都重跑二项式展开。把它记进结论清单里，用时直接调用；但要理解"这条结论的来源是第 1-5 步的严格推导，不是记忆速记"。
结论要"能推"而不是"能背"：如果哪天你怀疑这条结论——比如遇到 \(n\) 是分数或负数的情形——回到第 1-5 步的推导，你会发现每一步的逻辑对分数 / 负数也成立（需要广义二项式定理，进阶时展开）。这就是"推导 vs 记忆"的分野：推导能延伸，记忆只能覆盖背过的场景。
常用导数表（直接用）
\((C)' = 0\)（常数函数的导数为 0）
\((x)' = 1\)（一次函数导数为 1，因 \(n = 1\) 时 \((x^1)' = 1 \cdot x^0 = 1\)）
\((x^2)' = 2x\) · \((x^3)' = 3 x^2\) · \((x^n)' = n x^{n-1}\)
线性法则：\((a f + b g)' = a f' + b g'\)（常数提前、加法分配）。例：\((3 x^2 + 5 x)' = 6 x + 5\)。

卡 2 · 定积分 = 曲线下面积：用 v-t 图割条求和推 ∫ xn dx

▼
学习目标：从 v-t 图的几何直觉出发，理解"定积分 = 曲线下面积"；并用反求导直觉接受 \(\int x^n\,dx = \dfrac{x^{n+1}}{n+1} + C\)（\(n \neq -1\)）。

STEP 1几何动机
v-t 图的位移

先回到一个具体问题：已知速度 \(v(t) = 2 t\)（即从静止开始的匀加速运动，\(a = 2\) m/s²）。求 \(t = 0\) 到 \(t = 3\) s 的位移。

匀速时，位移 = 速度 × 时间——这等于 v-t 图上"矩形"的面积。
变速时，速度每秒都变——v-t 图不再是矩形，"位移 = 面积"这一直觉还成立吗？

下面的图把时间轴 \([0, 3]\) 切成 6 个等宽小段，每段宽 \(\Delta t = 0.5\) s，每段内把速度当作常数（取段末速度近似），算出 6 个小矩形面积之和，作为对真实位移的近似值。

SVG 1 · v(t) = 2t 的 v-t 图，时间区间 [0, 3] s 被切成 6 段；蓝色直线是真实速度，6 个浅蓝矩形是右端点近似（每条高度取该段右端速度）。
用 6 条矩形求位移近似值
第 i 段（i = 1, 2, …, 6）的右端时间是 \(t_i = 0.5 i\)，速度近似为 \(v(t_i) = 2 \cdot 0.5 i = i\) m/s。
第 i 段位移近似 \(\approx v(t_i) \cdot \Delta t = i \times 0.5\) m。
6 段相加：\(\sum_{i=1}^{6} i \times 0.5 = 0.5 \times (1+2+3+4+5+6) = 0.5 \times 21 = 10.5\) m（偏大，因为每段取了右端最大速度）。
如果改用左端近似：\(0.5 \times (0+1+2+3+4+5) = 7.5\) m（偏小）。真实位移介于二者之间。
STEP 2把细条变得无穷细
取极限

左端近似 7.5、右端近似 10.5——真实值在哪里？让条数 n 越来越多、每条 \(\Delta t\) 越来越小，看近似值收敛到多少。

细分到 n 条 · 取极限
把 \([0, 3]\) 分成 n 等份，每条 \(\Delta t = 3/n\)。第 i 段右端 \(t_i = 3 i / n\)，速度 \(v(t_i) = 2 \cdot 3 i / n = 6 i / n\)。
总和 \(S_n = \displaystyle\sum_{i=1}^{n} \frac{6 i}{n} \cdot \frac{3}{n} = \frac{18}{n^2} \sum_{i=1}^{n} i = \frac{18}{n^2} \cdot \frac{n(n+1)}{2} = \frac{9 (n+1)}{n} = 9 + \frac{9}{n}\)。
（用到等差数列求和公式 \(1 + 2 + \cdots + n = n(n+1)/2\)；如未学过此公式，可先接受这一中间结果，本课重点是"取极限 \(n \to \infty\)"这一步。）
当 \(n \to \infty\)：\(\displaystyle \lim_{n \to \infty} S_n = 9 + 0 = 9\) m。
结论：v(t) = 2t 从 0 到 3 s 的真实位移就是 9 m。
几何对照：v-t 图里 \([0, 3]\) 下方是一个底 = 3、高 = 6 的直角三角形，面积 \(= \tfrac{1}{2} \times 3 \times 6 = 9\) m。完全一致！
思考点 1 · "位移 = 面积"为什么是普遍真理
上面这段算的是 v(t) = 2t 的具体情形。但割条求和→取极限这一思路不依赖速度公式的具体形式——任何 v(t) 都能这样切。请你尝试用一句话说出："为什么 v-t 图下方的面积一定等于位移？"
我感觉是因为"每个小矩形的面积就是一小段位移"。请不要直接评判我的回答对不对，反过来问我：那"小矩形的面积"具体写出来是哪两个量的乘积？它和"位移"又是哪两个量的乘积？两者为什么一定相等？
在草稿上写："小矩形面积 = ___ × ___；小段位移 = ___ × ___；两者相等是因为___。"
关键词：宽 = Δt、高 = v、小段位移 = v · Δt。在 Δt → 0 的极限下，"速度不变"的近似变成精确。
定积分的正式定义
把上述"分割 → 近似 → 求和 → 取极限"这一过程取个名字，就得到定积分：
\[ \int_a^b f(x)\,dx \;\stackrel{\text{def}}{=}\; \lim_{n \to \infty} \sum_{i=1}^{n} f(x_i)\,\Delta x \]
符号读法：\(\int\) 是拉长的 S（Sum 求和的首字母）；\(a, b\) 是积分下限和上限；\(f(x)\) 是被积函数；\(dx\) 是无穷小宽度。
几何意义：\(\int_a^b f(x)\,dx\) 就是 \(y = f(x)\) 与 x 轴在 \([a, b]\) 围成的有向面积（\(f(x) > 0\) 处面积为正，\(f(x) < 0\) 处为负）。
物理意义（本课最关心的版本）：\(\displaystyle x = \int_0^T v(t)\,dt\)——v-t 图下方面积就是位移。
STEP 3∫ xn dx 的公式
反求导直觉

有了"定积分 = 面积"的语义后，下一个问题是：具体怎么算？ 总不能每道题都重新割条求和。这里要引入不定积分（原函数）。

原函数与不定积分
原函数：若 \(F'(x) = f(x)\)，称 \(F(x)\) 是 \(f(x)\) 的一个原函数（antiderivative）。
由卡 1 知 \((x^2)' = 2x\)，所以 \(x^2\) 是 \(2x\) 的一个原函数；\(x^2 + 5\) 也是（常数导数 = 0）。
不定积分：把 \(f(x)\) 的所有原函数写成 \(F(x) + C\)（\(C\) 为任意常数），记作 \(\displaystyle \int f(x)\,dx = F(x) + C\)。
思考点 2 · 反过来找 \(\int x^n\,dx\) · 用求导反推是否够严格
问题：要找一个函数 \(F(x)\)，让 \(F'(x) = x^n\)。卡 1 已推得 \((x^n)' = n\,x^{n-1}\)——如果我把这个结论"反过来看"：什么样的函数 \(F(x)\)，求一次导后能凑出 \(x^n\)？这一步是"猜 + 验证"还是"严格证明"？
"不要直接告诉我答案。请引导我探究：我猜 \(F(x) = \dfrac{x^{n+1}}{n+1}\)——但'猜' 本身不构成证明。我能做的最强的验证是什么？如果我对 \(\dfrac{x^{n+1}}{n+1}\) 求一次导，得到 \(x^n\)，这算不算证明了 \(\int x^n\,dx = \dfrac{x^{n+1}}{n+1} + C\)？"
对回卡 1 结论 \((x^k)' = k\,x^{k-1}\)，代 \(k = n+1\)，看看 \(\left(\dfrac{x^{n+1}}{n+1}\right)' = ?\)
在草稿上验证：\(\displaystyle\left( \frac{x^{n+1}}{n+1} \right)' = \frac{(n+1)\,x^{n}}{n+1} = x^n\)。✓ ——但这只证明了 "我这个猜测的 \(F\) 确实是原函数之一"，还不构成"\(\int x^n\,dx\) 的严谨推导"。
幂函数不定积分 · 反求导得到的通用结论
\[ \boxed{\,\int x^n\,dx = \frac{x^{n+1}}{n+1} + C \quad (n \neq -1)\,} \]
这是一条通用结论：来自"对 \(\dfrac{x^{n+1}}{n+1}\) 求导恰好得到 \(x^n\)"这一反求导直觉的验证。可迁移到任何幂函数积分，是下面公式推导的直接工具——用时直接调用，不需要每次都重新反求导。
为什么 \(n = -1\) 例外？若 \(n = -1\)，公式要除以 \(n + 1 = 0\)，分母为零。\(\int x^{-1}\,dx\) 的答案是 \(\ln |x| + C\)——本课不展开，记住"\(n = -1\) 除外"即可。
常数为什么不能丢？因为 \((x^{n+1}/(n+1) + 7)' = x^n\)、\((x^{n+1}/(n+1) - 100)' = x^n\)，加任何常数都满足。故必须写 \(+ C\)——这是"原函数不唯一"的自然反映。
严谨推导的进阶提示

上面的思路——反求导直觉 + 求导验证——本课够用，学生能理解并使用这条结论。但要把它称为"严谨推导"还差两步：

  1. "面积法的极限证明"：直接从定积分定义 \(\int_a^b x^n\,dx = \lim_{n \to \infty} \sum f(x_i)\Delta x\) 出发，通过 Riemann 和的极限严格算出 \(\dfrac{b^{n+1} - a^{n+1}}{n+1}\)。这需要处理"\(\sum_{i=1}^{n} i^k\)"的封闭形式（\(k = 1\) 时用等差数列、\(k = 2\) 时用二次求和公式、更高 \(k\) 需要 Bernoulli 数）。
  2. "牛顿-莱布尼茨定理"：证明"求导"和"积分"是一对严格的逆运算——即 \(F'(x) = f(x) \Rightarrow \int_a^b f\,dx = F(b) - F(a)\)。这一步把"反求导直觉"变成严密定理。定理的完整证明在下一课（牛顿-莱布尼茨公式 · 进阶）展开。

本课的"反求导直觉 + 求导验证"是够用且合理的教学路径——先接受结论、学会使用，然后进阶时再回来把每一步严谨补齐。知道这条结论有更严谨的推导路径在等着你，本身也是一种严谨。

把"工具"与"几何"接上 · 牛莱公式（仅做工具一句话）
把不定积分用到定积分上，可以用一条公式直接算出曲线下面积：\(\displaystyle \int_a^b f(x)\,dx = F(b) - F(a)\)，其中 \(F\) 是 \(f\) 的任一原函数。
这条公式叫牛顿-莱布尼茨公式，是微积分的"基本定理"。它的证明留到第 4 课，本课只把它当工具用一次：\(\displaystyle \int_0^3 2t\,dt = [t^2]_0^3 = 9 - 0 = 9\) m，与 STEP 2 割条极限的结果一致。
本课承诺：你不需要知道它为什么对，下一课会从积分上限函数 \(S(x) = \int_a^x f(t)\,dt\) 出发严格证明。

卡 3 · 推导公式 ① · v = v₀ + at（速度-时间关系）

▼
学习目标：从加速度的定义 \(a = \dfrac{dv}{dt}\) 出发，用积分一步推出 \(v = v_0 + at\)。
建议时长：35-40 分钟（前置 5 + 推导 15 + 物理含义 + checkpoint 20）
STEP 1前置回顾
a 的定义

来自第 1 课 · 运动的描述：

加速度是速度对时间的瞬时变化率：\(\displaystyle a(t) = \lim_{\Delta t \to 0} \frac{\Delta v}{\Delta t} = \frac{dv}{dt}\)。

匀加速直线运动的前提：\(a\) 是常数。

来自卡 1-2：

已知 \((C)' = 0\) 与 \((x^n)' = n x^{n-1}\)；已知 \(\int x^n\,dx = \dfrac{x^{n+1}}{n+1} + C\)。

把 a 当成 0 次幂：\(a = a \cdot t^0\)，于是 \(\int a\,dt = a t + C\)（常数提到积分外）。

STEP 2两步推导
先自己写
思考点 1 · 自己推 v(t)
起点：\(a = \dfrac{dv}{dt}\)。要得到 \(v\) 对 \(t\) 的函数。提示：把 \(dt\) 移到等式右边、两边各算一个不定积分。注意"初始条件"：\(t = 0\) 时 \(v = v_0\)。
我推到 \(v = a t + C\) 这一步卡住了，不知道 C 怎么定。请不要直接给答案，反过来问我：t = 0 时方程两边各是什么？由此能不能解出 C？
在草稿上写出 3 行：① 由定义；② 两边积分；③ 用初始条件 \(v(0) = v_0\) 解 C。
推导 · v = v₀ + at
① 由加速度定义：\(\dfrac{dv}{dt} = a\)（a 是常数）
② 两边对 t 不定积分：\(\displaystyle \int \frac{dv}{dt}\,dt = \int a\,dt \;\Rightarrow\; v = a t + C\)
③ 用初始条件定 C：当 \(t = 0\)，\(v = v_0\)，代入得 \(v_0 = a \cdot 0 + C\)，即 \(C = v_0\)。
结论：\(\displaystyle \boxed{\,v = v_0 + a t\,}\)。
等价的定积分形式（从 0 到 t）：\(\displaystyle \int_{v_0}^{v} dv' = \int_0^t a\,dt' \;\Rightarrow\; v - v_0 = a t\)，结果相同。
STEP 3物理含义
读公式
  • \(v_0\) 是 \(t = 0\) 时刻的速度——"起跑线上的速度"。
  • \(a t\) 是速度的增量——每秒增 \(a\) m/s，过了 \(t\) 秒共增 \(a t\) m/s。
  • 等号成立的前提：\(a\) 是常数（匀加速）。若 \(a\) 随时间变化，需要重新积分 \(v = v_0 + \int_0^t a(t')\,dt'\)。
  • 方向：\(v, v_0, a\) 都是有方向的量。一维直线运动里，先定一个"正方向"（如向右），其余量按正负代入即可——刹车时 \(a < 0\)、自由下落时取下为正则 \(a = +g\)。

卡 4 · 推导公式 ② · x = v₀ t + ½ a t²（位移-时间关系）

▼
学习目标：用 \(v = \dfrac{dx}{dt}\) 与卡 3 的结论 \(v = v_0 + a t\) 联立，对时间积分推出位移公式；并理解 \(\dfrac{1}{2}\) 的几何来源。
建议时长：35-40 分钟（推导 15 + 几何意义 + checkpoint 20）
STEP 1推导 · 代数路径
v = dx/dt → 积分
思考点 1 · 自己推 x(t)
起点：\(v = \dfrac{dx}{dt}\)。把卡 3 的 \(v = v_0 + a t\) 代入，对 t 不定积分。初始条件：\(t = 0\) 时 \(x = 0\)（位移从起点算起）。
我得到 \(x = v_0 t + ? \cdot t^2 + C\)，但不确定 t² 前面的系数。请不要给答案，反过来问我："\(\int a t\,dt\) 应该套幂函数积分公式中的哪一条？n 是几？"
在草稿写 4 行：① \(v = dx/dt\)；② 代入卡 3 结论；③ 两边对 t 积分；④ 用初始条件定 C。
关键：\(\int a t\,dt = a \int t^1\,dt = a \cdot \dfrac{t^2}{2} = \dfrac{1}{2} a t^2\)。
推导 · x = v₀ t + ½ a t²
① 由速度定义：\(\dfrac{dx}{dt} = v\)
② 代入卡 3 结论：\(\dfrac{dx}{dt} = v_0 + a t\)
③ 两边对 t 不定积分：\(\displaystyle x = \int (v_0 + a t)\,dt = v_0 \int dt + a \int t\,dt = v_0 t + \tfrac{1}{2} a t^2 + C\)
④ 用初始条件定 C，认出它的物理含义：当 \(t = 0\) 时，物体所在的位置就是起始位置，记为 \(x_0\)。代入上式：\(x_0 = v_0 \cdot 0 + \tfrac{1}{2} a \cdot 0^2 + C\)，得 \(\boxed{C = x_0}\)——积分常数 \(C\) 就是起始位置。这是数学结构里 \(C\) 的物理身份。
⑤ 通式 + 约定：通式是 \(\displaystyle x = x_0 + v_0 t + \tfrac{1}{2} a t^2\)。在本课与后续大多数直线运动问题里我们约定坐标原点取在物体的起始位置（也就是 \(x_0 = 0\)）——此时通式塌缩为：
结论：\(\displaystyle \boxed{\,x = v_0 t + \tfrac{1}{2} a t^2\,}\)（\(x_0 = 0\) 的约定下）。如果题目把原点放在别处，就要换回带 \(x_0\) 的通式。
STEP 2推导 · 几何路径
v-t 图梯形面积

同一个公式也可以从 v-t 图直接读出来——这是"½ 从哪来"的几何答案。

SVG 2 · 匀加速 v-t 图：蓝色直线 v = v₀ + at；下方梯形面积 = 位移。梯形被一条金色虚线分成"下矩形 v₀·t"和"上三角形 ½·at·t = ½ a t²"。
几何解释 · "½" 是三角形面积的来源
v-t 图下方在 \([0, t]\) 区间是一个梯形，上底 \(v_0\)、下底 \(v_0 + a t\)、高 = t。
把梯形切成"下矩形 + 上三角形"两块：
　• 下矩形：宽 \(t\) × 高 \(v_0\) = \(v_0 t\)（"如果一直以 \(v_0\) 走"贡献的位移）
　• 上三角形：底 \(t\) × 高 \(a t\) × \(\tfrac{1}{2}\) = \(\tfrac{1}{2} a t^2\)（"加速带来的额外位移"）
两块相加就是 \(x = v_0 t + \tfrac{1}{2} a t^2\)——和代数推导完全一致。
"½" 的物理意义：它不是从天上掉下来的，是三角形面积公式里的 ½。换言之，幂函数 \(t\) 的积分公式 \(\int t\,dt = \tfrac{1}{2} t^2\) 和"三角形面积 = ½ × 底 × 高"是同一件事。

卡 5 · 推导公式 ③ · v² = v₀² + 2ax（速度-位移关系）

▼
学习目标：用代数消去 \(t\)、或用 \(a\,dx = v\,dv\) 直接积分两条路径，推出不显含 t 的公式 \(v^2 = v_0^2 + 2 a x\)。
建议时长：35-40 分钟（代数法 15 + 微积分法 15 + checkpoint 10）
STEP 1代数路径
消去 t

把卡 3 和卡 4 已经推出的两条公式放在一起（约定：\(x\) 是从初始位置算起的位移，\(x_0 = 0\)）：

① \(v = v_0 + a t\)
② \(x = v_0 t + \tfrac{1}{2} a t^2\)

这两条公式都含 \(t\)。如果题目不告诉时间 \(t\)，只给 \(v_0, a, x\) 求 \(v\)（或反过来），就要把 \(t\) 消掉。

思考点 1 · 消去 t
从公式 ① 把 \(t\) 解出来：\(t = (v - v_0)/a\)。代入公式 ②，再化简。建议先在草稿上把代入的式子写完整、再化简，不要心算。
我代入之后展开得到一堆带 \(v_0 v\) 与 \(v_0^2\) 的项，化不开。请不要给中间式，反过来让我"把所有项乘以 2a，转成一行不含分式的等式"。
3 步：① 由 ① 解 t；② 代入 ②；③ 两边乘 2a 化简。
化简关键：\((v - v_0)^2 = v^2 - 2 v_0 v + v_0^2\)；最后会发现 \(v_0 v\) 项相互抵消，只剩 \(v^2 - v_0^2\)。
代数推导 · v² = v₀² + 2ax
① 从 \(v = v_0 + a t\) 解 \(t\)：\(t = (v - v_0)/a\)。
② 代入 \(x = v_0 t + \tfrac{1}{2} a t^2\)： \[ x = v_0 \cdot \frac{v - v_0}{a} + \tfrac{1}{2} a \left( \frac{v - v_0}{a} \right)^2 \]
③ 化简（两边乘 \(2a\)）： \[ 2 a x = 2 v_0 (v - v_0) + (v - v_0)^2 \] 展开：\(= 2 v_0 v - 2 v_0^2 + v^2 - 2 v_0 v + v_0^2 = v^2 - v_0^2\)。
结论：\(\displaystyle \boxed{\,v^2 = v_0^2 + 2 a x\,}\)。
STEP 2微积分路径
a dx = v dv

同样的结论也可以不经过 t 直接得到——这就是链式法则的妙用。

链式法则路径
① 把 a 改写：\(\displaystyle a = \frac{dv}{dt} = \frac{dv}{dx} \cdot \frac{dx}{dt} = v \cdot \frac{dv}{dx}\)（用了 \(v = dx/dt\)）。
② 移项：\(a\,dx = v\,dv\)。
③ 两边定积分（左：从 0 到 x；右：从 \(v_0\) 到 \(v\)）： \[ \int_0^x a\,dx' = \int_{v_0}^v v'\,dv' \;\Rightarrow\; a x = \tfrac{1}{2}(v^2 - v_0^2) \]
④ 整理：\(v^2 - v_0^2 = 2 a x\)，即 \(v^2 = v_0^2 + 2 a x\)。和代数法完全一致。
思考点 2 · 两条路径的味道
代数法的过程是"把 t 解出来再代入再化简"，微积分法是"先把 dt 换成 dx/v 再积分"。哪条更"工程"？哪条更"数学"？
我觉得代数法更直接但化简琐碎，微积分法更优雅但要先想到链式法则。请不要直接评判，反过来问我："如果题目里 a 不是常数（比如 \(a = -k x\)，弹簧情形），上面哪条路径还行得通？"
写一句话："a 为常数时两路径等价；a 随 x 变化时，更通用的是____路径。"
答案是微积分路径。因为 \(a\,dx = v\,dv\) 对 a 是否常数都成立——只要把左边按 \(a(x)\,dx\) 积分即可，而代数消 t 法只在 a 为常数时灵光。

本课从 (xn)' = n xn-1 一条法则出发，连续推出三条匀加速直线运动公式。读到这里，把"种子"和"果实"的关系画一遍。

回顾 A"种子" → 三公式
一条主链路
主链路 · 一句话版
"加速度是速度的导数" + "速度是位移的导数"——这是两条物理事实；
"幂函数 \(t^n\) 的积分是 \(t^{n+1}/(n+1)\)"——这是一条数学工具；

三公式对照表
① v = v₀ + at——速度的"加成 + 累积"。已知时间求当前速度。
② x = v₀t + ½ a t²——位移的"匀速分量 + 加速分量"。已知时间求当前位移。
③ v² = v₀² + 2ax——速度与位移的代数关系（不显含时间）。已知位移求速度。
选用规则：题面给什么、求什么，选对应公式即可。三公式之间通过 t 这把"梭子"互相穿连。
回顾 B为什么
不用背？推导链路在脑中

把"背三公式"换成"会推三公式"，不是为了考试更难一点，而是为了下面三件事：

  • 当 a 不再是常数时（弹簧、阻力、变力运动），三公式不再适用；但推导链路依旧——\(\int a(t)\,dt\) 和 \(\int v(t)\,dt\) 永远成立。"会推"的人能扩展，"会背"的人卡住。
  • 当题目方向变复杂时（自由落体、抛体的水平/竖直分解），公式里 \(v_0, a, x\) 都是矢量，正负要靠自己定。"会推"的人记得初始条件的位置，"会背"的人容易代错符号。
  • 它是后续学习的基石：圆周运动、抛体运动、牛顿动力学都要重复"由 a 积分 → v → x"这个模板。本课练熟一次，后面省力百次。
